## Overview

This guide describes a three-phase approach to implementing Agent Experience (AX) principles in an organization or product. Note: this is the **broader "Agent Experience" movement** initiated by Netlify, distinct from (but conceptually aligned with) Kun Chen's **AXI** framework at axi.md.

## Three-Phase Approach

**Phase 1 — Assessment:**

* Survey agents
* Review existing tools
* Analyze interaction metrics
* Identify pain points

**Phase 2 — Planning:**

* Define improvement roadmap
* Set success metrics
* Establish timeline and resource allocation

**Phase 3 — Implementation:**

* Tool optimization
* Process improvement
* Training/onboarding development

## Key Principle

> "Improving agent experience is an ongoing process. Regular assessment and adjustment are key to long-term success."
